centos7 通过docker 安装 mysql

1、通过docker查询mysql镜像：docker search mysql 如图：

2、下载自己想要的版本，我下载的是5.7的 如图：

3、查看下载是否成功 执行docker images 如图：

4、创建并启动容器：

-p 3306:3306 :将容器的3306端口映射到主机的3306端口
–name mysql： 容器别名
-v /lisanjin9/mysql/conf:/etc/mysql/ \ ：将配置文件夹挂载到主机
-v /lisanjin9/mysql/log:/var/log/mysql \ ：将日志文件夹挂载到主机
-v /lisanjin9/mysql/data:/var/lib/mysql \ ：将配置文件夹挂载到主机
-e MYSQL_ROOT_PASSWORD=root \ ：初始化root用户的密码
-d mysql:5.7以后台方式启动，以mysql:5.7镜像启动
此处的挂在是在根目录下(/)创建的文件夹 lisanjin9

4、查看docker正在运行的镜像：docker ps 如图 :

5、使用命令 docker exec -it mysql /bin/bash 进入容器内部：

6、使用命令 mysql -uroot -p 登录mysql

7、设置mysql跟随docker重启，自动启动 ： docker update mysql –restart=always

  • 通过docker启动mysql ：docker start mysql
  • 通过docker停止mysql ：docker stop mysql
  • 通过docker重启mysql ：docker restart mysql
